### Run Locally

To run **ConvertMoney** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/daianaadepaula/update-convert-money.git
```

Windows:

```bash
git clone https://github.com/daianaadepaula/update-convert-money.git
```
